## Requirements

- Flutter `>=3.0.0`
- Dart `>=3.0.0`

## Add the package

```bash [Terminal]
flutter pub add feda_flutter
```

Or add it manually to your `pubspec.yaml`:

```yaml [pubspec.yaml]
dependencies:
  feda_flutter: ^0.2.0
```

Then run:

```bash [Terminal]
flutter pub get
```

## Android setup

If you use `PayWidget` (WebView), add internet permission to `AndroidManifest.xml`:

```xml [AndroidManifest.xml]
<uses-permission android:name="android.permission.INTERNET" />
```

## iOS setup

No additional setup required.
